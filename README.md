# TechZone — E-Commerce Store

A fully functional e-commerce website for an electronics and gadgets store, featuring a complete shopping cart system built with vanilla JavaScript.

## 🔗 Live Demo
[Visit Live Site](https://mehr-un-nisa07.github.io/techzone/)

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript (with LocalStorage for cart persistence)
- GitHub Pages (Hosting)

## ✨ Features
- Modern dark-themed UI with cyan accent design
- Fully responsive across desktop, tablet and mobile
- **Working shopping cart** — add to cart, change quantity, remove items
- Cart data persists using browser LocalStorage
- **Live product filtering** — filter by category, price range, sale items, new arrivals, and minimum rating
- **Sorting** — sort products by price (low/high) or rating
- Coupon code system on checkout (try code: `TECHZONE10`)
- Flash sale countdown timer
- Newsletter subscription box
- FAQ accordion on contact page

## 📄 Pages
- **Home** — Hero banner, category grid, featured products, flash sale, testimonials
- **Products** — Full catalogue with sidebar filters and sorting
- **Cart** — Cart management with order summary and checkout flow
- **About** — Company story, values, team
- **Contact** — Contact form and FAQ accordion

## 🎯 Purpose
Built as a portfolio project to demonstrate JavaScript skills beyond static pages — including state management with LocalStorage, dynamic DOM manipulation, and real e-commerce UX patterns like filtering, sorting, and cart logic.
